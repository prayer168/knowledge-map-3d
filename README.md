# Marble Skill Taxonomy 3D｜雙語知識地圖

Interactive 3D Traditional Chinese / English knowledge graph built from the [Marble Skill Taxonomy](https://github.com/withmarbleapp/os-taxonomy).

The graph includes all 1,590 learning topics and 3,221 prerequisite relationships. It supports orbit, zoom, bilingual search, subject filters, relationship highlighting, and prerequisite/unlock inspection.
